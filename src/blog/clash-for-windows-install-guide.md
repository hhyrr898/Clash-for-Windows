## 安装前确认

安装前建议确认系统版本、网络状态和本机是否已有同类客户端运行。若电脑上保存过旧配置，可以先将配置目录复制一份，后续出现异常时便于恢复。

## 安装步骤

下载 Windows 版后，按安装向导完成基础安装。首次启动时留意系统提示，如果需要网络权限，请根据系统弹窗确认。进入主界面后，先不要急着切换策略，建议先完成配置导入。

## 导入配置

在 Profiles 页面填写订阅地址并更新，随后进入 Proxies 页面选择合适策略。若列表为空，可以查看日志中是否存在地址错误、网络超时或文件格式异常。

## 基础检查

确认系统代理开关、端口状态和浏览器访问表现。若部分应用无法使用，可查看该应用是否单独设置了代理，或是否被安全软件限制。
