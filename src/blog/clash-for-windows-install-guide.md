## 安装前确认

安装前先确认三件事：系统是 Windows 10/11，电脑时间自动同步，旧版客户端或同类代理工具已经退出。若电脑上保存过旧配置，可以先把配置目录复制一份，后续出现订阅更新失败或策略组为空时便于恢复。

如果浏览器提示下载文件不常见，不要直接忽略。先确认下载入口、文件名和保存位置，再决定是否继续安装。公司电脑还要留意安全软件是否会拦截网络驱动或代理端口。

## 安装步骤

下载 Windows 版后，按安装向导完成基础安装。首次启动时留意 Windows 安全提示，如果需要网络权限，请根据系统弹窗确认。进入主界面后，先不要急着切换策略，建议按顺序完成配置导入、系统代理检查和日志确认。

安装完成后可以先看托盘图标是否出现。图标存在但主界面打不开时，通常是程序仍在后台运行，可以从托盘右键退出后重新打开。

## 导入配置

在 Profiles 页面填写订阅地址并更新，随后进入 Proxies 页面查看策略组是否正常显示。若列表为空，优先看日志中是否存在地址错误、网络超时、证书异常或文件格式异常。

建议给配置起一个能看懂的名字，例如“主力订阅-2026-06”。多个配置混在一起时，后面排查会省很多时间。

## 基础检查

确认系统代理开关、端口状态和浏览器访问表现。若部分应用无法使用，可查看该应用是否单独设置了代理，或是否被安全软件限制。

第一次测试不要同时打开多个代理软件。先用浏览器访问常用页面，再测试终端、开发工具或其他桌面软件。只有某个应用异常时，问题通常在该应用自己的代理设置里。
