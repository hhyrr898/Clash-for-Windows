## 权限确认

macOS 环境中需要留意系统网络权限和安全提示。首次运行时如果出现授权弹窗，应按提示完成确认。

## 配置导入

导入订阅后，查看策略组是否完整。若配置无法下载，先检查系统时间和网络状态，再确认订阅地址是否有效。

## 桌面习惯

可根据使用频率决定是否加入登录项。若只是偶尔使用，手动启动会更容易掌握当前状态。
