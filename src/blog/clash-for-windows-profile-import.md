## 添加订阅地址

打开 Profiles 页面，将服务提供方给出的订阅地址粘贴到输入框中并执行下载。命名时建议使用日期或用途，避免多个配置混在一起无法区分。

## 更新与切换

配置更新后，可以回到 Proxies 页面查看策略组是否正常显示。多个配置之间切换时，建议保留最近可用的一份作为备用。

## 备份建议

若经常在多台设备之间使用，可以将配置文件夹定期备份。迁移时只复制必要文件，避免把旧日志和缓存一并带入新设备。
